# 🚀 Terminal Agents

Enterprise-grade autonomous AI agent ecosystem for software development.

## Quick Start

`ash
# Activate virtual environment
.\venv\Scripts\Activate.ps1

# Run demo
python -m terminal_agents.cli demo

# Execute a task
python -m terminal_agents.cli execute "Your task here"
`

## Features

✓ Multiple specialized agents
✓ Task orchestration
✓ Memory system
✓ Tool execution
✓ Production-ready code

## Installation

1. Create virtual environment: python -m venv venv
2. Activate: .\venv\Scripts\Activate.ps1
3. Install dependencies: pip install -r requirements.txt
4. Create .env with your API key
5. Run: python -m terminal_agents.cli demo
